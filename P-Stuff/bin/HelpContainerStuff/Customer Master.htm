CustomerMaster:

 

Details of the Customer are being entered into CustomerMaster, where Customer Code is auto-generated.

 

Add: This option is used for adding a new record in the list.

 

Edit: For editing a record which already exists in the list, edit option can be used.

 

Delete: Delete option is used for deleting a particular record from the list of records.

 

Exit: Exit option is used for exiting from this particular form.

 

For navigation through all the records, options like First, Next, Previous & Last are being provided.

 

Update: Update option is being provided to update a newly entered record or update the edited record. After update button is being clicked, the record is being updated and the last record in the list is being displayed.

 

Refresh: Refresh or Cancel option is used for refreshing a record or for canceling the changes which should not be updated.

 

Details like Customer Name, Address, City, TelNo., Mobile No., Pin Code needs to be entered manually. Country can be selected from dropdown list which is being provided.
