ContainerMaster:

 

Container Master can be used for adding details about a particular Container. Its Length, Width, Height can be entered accordingly.

 

Add: This option is used for adding a new record in the list.

 

Edit: For editing a record which already exists in the list, edit option can be used.

 

Delete: Delete option is used for deleting a particular record from the list of records.

 

Exit: Exit option is used for exiting from this particular form.

 

For navigation through all the records, options like First, Next, Previous & Last are being provided.

 

Update: Update option is being provided to update a newly entered record or update the edited record. After update button is being clicked, the record is being updated and the last record in the list is being displayed.

 

Refresh: Refresh or Cancel option is used for refreshing a record or for canceling the changes which should not be updated.

 

In Container Master, Container No. is being auto-generated for every new entry which is being made. Value for a particular Container is entered in Kgs. and is automatically converted into Lbs..

 

Size option is further divided into Length, Width and Height where measurements are entered in the form of Feet, Inches, Cms., and Mms..

 

After all the measurements are entered in their respective fields, Total Size in Cu. Ft. and Cu. Mt. are automatically calculated.

 

Tab Index as well as Enter Key options both have been provided according to the convenience of the user. HotKey combinations have been provided wherever essential.
